# Prueba-Tecnica-DDB-Vue

Desarrollo de la prueba técnica

## Tecnologías usadas
- TypeScript
- Sass (Scss)
- Pinia


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
